**Transform internal monologues into execution systems.**

Yourfocus is an integrated ecosystem designed for active knowledge management based on Kaizen principles. It helps you capture insights through voice memos, manage tasks via AI-assisted workflows, and maintain deep focus with integrated Pomodoro analytics.

## üåü Key Features

- **Yourfocus Timer**: Pomodoro intervals with automatic tracking and analytics.
- **Voice Bit√°cora**: Capture ideas on the go with high-quality audio recording and AI-powered transcription.
- **AI Agent (Brain)**: A personal assistant that structures, connects, and automates your personal knowledge base.
- **Zettelkasten / Knowledge Base**: Seamless integration with tools like Obsidian for long-term memory.
- **BYOK (Bring Your Own Key)**: Use your own AI provider keys to scale intelligence limits.
- **Tasks & Kanban**: Auto-generated task boards derived from your voice logs.

## üöÄ Getting Started

### Prerequisites

- Node.js (v18+)
- Bun
- Supabase CLI

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   bun install
   ```
3. Set up environment variables:
   ```bash
   cp .env.example .env
   ```
4. Start the development server:
   ```bash
   bun dev
   ```
