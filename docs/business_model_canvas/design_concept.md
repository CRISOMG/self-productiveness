Concepto: **_Yourfocus_**

Estilo Visual (The Kurzgesagt-Flat Framework):

Geometría: Ilustración vectorial plana (Flat Design) con formas geométricas redondeadas y curvas suaves.

Composición: Minimalismo detallado con formas simplificadas pero de máximo impacto visual. Sin contornos (sin line art).

Color: Bloques de color sólido (Color Blocking) con saturación extremadamente alta. Contraste dramático entre tonos cálidos y fríos para generar tensión visual energética.

Paleta de Identidad (Valores Yourfocus):

#DB6C03 (Naranja): Energía y Autoagencia.

#174909 (Verde): Sistemas y Autorregulación.

#602D1F (Marrón): Profundidad y Verdad Radical (Lucidez).

#F7EFC5 (Crema): Claridad y Curiosidad Intelectual.

Fondo Base: Usa el Dark Slate/Green (el tono oscuro, aprox #0D222D) para dar profundidad y seriedad.

Ambiente: Estética Lofi/Cozy moderna. El resultado debe sentirse profesional, limpio y acogedor, como una pieza de divulgación científica de alta calidad.
